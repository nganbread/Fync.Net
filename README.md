# Fync.Net

A Dropbox/SkyDrive/OneDrive/Google Drive/Box alternative written in .NET and built to run in Azure

.NET components used
* .NET4.5
* OWIN
* OWIN Identity
* WebApi
* Entity Framework
* TinyIoC
* Moq
* NUnit
* WPF NotifyIcon
* Fody

Azure services used
* SQL Server
* Table Storage
* Blob Storage

Other components:
* SQLite
